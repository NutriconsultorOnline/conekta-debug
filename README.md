# conekta-debug
Conekta.js — for testing only. Downloads `conekta.js` from the Conekta CDN. 

## Installing
```sh
$ npm install conekta-debug
```

## Usage
With Browserify:

```js
var Conekta = require('conekta-debug');
```

Or `<script>` tags:

```html
<script src="./node_modules/conekta-debug/conekta-debug.js"></script>
```
